AjaxGroup

AjaxGroup.new()

Module: core/AjaxGroup

Extends: EventHandler

Events

openedconnection

Gets triggered if one of sub Ajaxs sends a request.

closedconnection

Gets triggered if one of sub Ajaxs gets respond.

openedfirstconnection

Gets triggered if one of sub Ajaxs sends a request and there isn't another opened request.

closedlastconnection

Gets triggered if one of sub Ajaxs gets respond and there isn't another opened request.

maxconnection

Gets triggered if there is no room for new connection.

Methods
