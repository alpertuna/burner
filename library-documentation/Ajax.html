Ajax

Ajax.new()

Module: burner/core/Ajax

Extends: EventHandler

Events

success

Gets triggered when respond is successful.

Fired by: Ajax#send

fail

Gets triggered when respond is fail.

Fired by: Ajax#send

always

Gets triggered when respond is got.

Fired by: Ajax#send

maxconnection

Gets triggered when bound ajax group has no room for new request.

Fired by: Ajax#send
